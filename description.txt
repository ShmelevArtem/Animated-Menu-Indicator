Animated Menu Indicator using CSS & Javascript

Небольшое красивое горизонтальное меню, в котором фон главного окна меняется при наведении курсора мыши.
Использовался html и css и javascript.

Оригинальное видео, чтобы повторить такое:
https://www.youtube.com/watch?v=xYNYGETrHJM

Мои изменения: меню переведено на русский язык; немного улучшен CSS код, для анимации кнопок; JavaScript вынесен в отдельный файл.
